# FudanLMS
### 身份与对应权限

在FudanLMS中共有4种身份，每种身份拥有不同的操作权限。

#### 通用权限

所有身份的用户均可：

- 使用FudanLMS通过书名、作者、ISBN查找相关书籍。

- 通过用户名和对应密码进行登录。

#### 暂停借阅的学生权限

被暂停借阅的学生还可以：

- 修改密码。
- 查询**自己**的借阅记录、还书期限。
- 查询**自己**是否有超期未还的书籍。
- 延长**自己**的还书期限。（增加延长次数）
- 归还**自己**借阅的书籍。

#### 正常借阅的学生权限

正常借阅的学生还可以：

- 借阅书籍。

#### 管理员权限

管理员用户还可以：

- 注册新的管理员账号或学生账号。
- 添加或删除数据库中的书籍。

- 重置**任何用户**的密码为默认值。
- 查询**任何用户**的借阅记录、还书期限。
- 查询**任何用户**是否有超期未还的书籍。
- 延长**任何用户**的还书期限（不增加延长次数）
